# PowerGo - Сайт для аренды генераторов

Современный, анимированный сайт для компании PowerGo, специализирующейся на аренде генераторов в Алматы.

## Особенности

### Анимации и эффекты:
- **Электрические разряды** - Canvas анимация с молниями и частицами на фоне
- **Переключатель темы** - Интерактивный свитч для смены светлой/темной темы с эффектом искры
- **Glitch эффект** - На главном заголовке
- **3D эффект карточек** - При наведении карточки генераторов наклоняются следуя за курсором
- **Пульсирующие индикаторы** - Анимированный индикатор питания в hero-секции
- **Плавная прокрутка** - Smooth scroll между секциями
- **Появление элементов** - Элементы появляются при прокрутке страницы
- **Parallax эффект** - Hero-секция двигается при прокрутке
- **Свечение курсора** - Эффект свечения вокруг курсора
- **Анимация молний** - Периодические вспышки молний на фоне

### Функциональность:
- Полностью адаптивный дизайн (мобильные устройства, планшеты, десктоп)
- Интеграция с WhatsApp для быстрой связи
- Прямые ссылки для звонков
- Информация о двух генераторах (Yuchai 125 kVA и Isuzu 55 kVA)
- Описание процесса работы (3 шага)
- Контактная информация
- Сохранение выбранной темы в localStorage

## Как запустить

1. Просто откройте файл `index.html` в браузере
2. Или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (с установленным http-server)
npx http-server
```

3. Откройте в браузере: `http://localhost:8000`

## Как изменить контент

### Изменить контактную информацию:

В файле `index.html` найдите и измените:
- Номер телефона: `+77076782151`
- WhatsApp ссылки: `https://wa.me/77076782151`

### Добавить новый генератор:

В файле `index.html` найдите секцию `.generators-grid` и скопируйте блок `.generator-card`, изменив:
- Название модели
- Мощность
- Характеристики
- Цену

### Изменить цвета:

В файле `styles.css` в разделе `:root` измените переменные:
```css
--primary-color: #fcda00;  /* Желтый акцент */
--bg-dark: #0a0a0a;        /* Темный фон */
--bg-card: #1a1a1a;        /* Фон карточек */
```

### Настроить интенсивность анимаций:

В файле `script.js` в классе `ElectricBackground`:
- Измените количество частиц (строка 13): `for (let i = 0; i < 50; i++)`
- Измените частоту молний (строка 29): `if (Math.random() < 0.3)`

## Технические детали

### Использованные технологии:
- HTML5 (Canvas API)
- CSS3 (Animations, Transitions, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Без зависимостей и фреймворков

### Совместимость:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Мобильные браузеры

### Оптимизация:
- Анимации используют `requestAnimationFrame` для плавности
- На мобильных устройствах интенсивность анимаций снижена
- Используется `localStorage` для сохранения темы
- Lazy loading для тяжелых элементов
